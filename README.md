# 7376231CS301

This repository contains my campus hiring evaluation work for the full stack notification system problem.

The goal is to design and build a simple, reliable notification platform where students can receive updates about placements, events, and results.

## Structure

```text
logging_middleware/
notification_system_design.md
notification_app_be/
notification_app_fe/
```

## Contents

- `logging_middleware/` contains the reusable logging helper used by the application.
- `notification_system_design.md` contains the design notes and stage-wise answers.
- `notification_app_be/` contains the backend service code.
- `notification_app_fe/` contains the frontend application.

## Run

Each application folder has its own README with the commands needed to install dependencies and run that part of the project.
